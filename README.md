# ❤️ Heart Disease Prediction using Machine Learning

A Machine Learning web application that predicts the likelihood of heart disease based on patient clinical information. The application is built using **Scikit-learn** and deployed with **Streamlit**.

## 📌 Features

* Predicts the likelihood of heart disease.
* User-friendly Streamlit interface.
* Displays prediction probability.
* Risk classification (Low / Medium / High).
* Feature importance visualization.
* Prediction confidence chart.
* Downloadable prediction report.

---

## 🛠️ Tech Stack

* Python
* Scikit-learn
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Streamlit
* Joblib
